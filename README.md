# Movie Listing and Rating Website

A responsive movie listing UI built from the PDF brief. The site includes:

- A cinematic landing section with a featured spotlight card
- Search, genre, year, and sorting controls
- Responsive movie cards with hover states
- A 1 to 5 star rating system saved in `localStorage`
- A movie details modal with cast, runtime, genres, and trailer links

## Run locally

This project is a static frontend, so no install step is required.

1. Open `index.html` in your browser.
